# Sorting Visualizer

An interactive web-based Sorting Visualizer built using HTML, CSS, and JavaScript. It helps users understand how different sorting algorithms work through animated visualizations.

## Features

- Generate random arrays
- Enter custom arrays manually
- Visualize sorting step-by-step
- Supports multiple sorting algorithms:
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
  - Merge Sort
  - Quick Sort
- Color-coded animations for each algorithm
- Sorting completion indicator

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How to Use

1. Generate a random array or enter a custom array.
2. Click **Use Array** if entering custom values.
3. Select any sorting algorithm.
4. Watch the sorting process animate in real time.
